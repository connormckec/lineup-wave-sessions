Watcher Settings

Configure these in Railway → Variables.

NTFY_TOPIC — optional server fallback for personal testing only
LOW_SLOTS_THRESHOLD — alert when slots ≤ this (default: 2)
CHECK_EVERY_MINS — check interval (default: 5)
MAX_SLOT_CHECKS — slot counts per cycle (default: 50)
SCRAPE_WEEKS_AHEAD — calendar lookahead in weeks (default: 4)
SMS_ADDRESS — carrier SMS email address
